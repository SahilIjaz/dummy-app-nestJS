# Backend (Nest.js)

## Overview
This is the backend API for the Dummy App, built with Nest.js.

## Scripts
- \`npm run start:dev\`: Start development server on localhost:3001
- \`npm run build\`: Build for production
- \`npm run test\`: Run unit tests
- \`npm run test:e2e\`: Run e2e tests

## Tech Stack
- Nest.js
- TypeScript
- Jest
