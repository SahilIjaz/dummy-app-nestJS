# Frontend (Next.js)

## Overview
This is the frontend application for the Dummy App, built with Next.js 15.

## Scripts
- \`npm run dev\`: Start development server on localhost:3000
- \`npm run build\`: Build for production
- \`npm run start\`: Start production server
- \`npm run lint\`: Run ESLint

## Tech Stack
- Next.js 15
- React 19
- TailwindCSS
- TypeScript
